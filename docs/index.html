Maximizing User Experience | Understanding Short-Term Memory in Website Interaction

Short-term memory is like a mental notepad with limited space. It can hold around 7 chunks of information but forgets them within 20 seconds if not rehearsed. To create user-friendly websites, design with these limits in mind.
